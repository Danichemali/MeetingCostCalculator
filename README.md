# Meeting Cost Calculator
*CSCI410 Mobile Application Development - Project 1*

A simple Android application that calculates the real-time cost of meetings based on attendees' salaries and meeting duration.

## Features
- Add/remove meeting attendees with custom hourly rates
- Real-time cost calculation
- Support for one-time, weekly, and monthly meetings
- Cost comparison between different meeting scenarios
- Pre-set common job roles for quick addition

## Technology Stack
- Android Studio
- Java
- XML for layouts

## Installation
1. Clone this repository
2. Open in Android Studio
3. Build and run on emulator or device
